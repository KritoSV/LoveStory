# Инструкция для работы с Markdown

## Выделение текста
 Что выделить текст курсивом, необходимо обрамить его звёздочками (*). 
 * Например: *Вот такк*

 Чтобы выделить текст полужирным, необходимо обрамить его двойными звёздочками (**).
 * Например **Вот так**
